# Example

Each example focuses on a single platform or concept and is designed to be easy to read, modify, and learn from.

| Example | File                   | Description                                                       |
|---------|------------------------|-------------------------------------------------------------------|
| CHIP-8  | [chip8.asm](chip8.asm) | A minimal **CHIP-8** program demonstrating raw opcode generation. |
| DOS     | [dos.asm](dos.asm)     | A tiny DOS `.COM` program using **MS-DOS** interrupts.            |
| WASM    | [wasm.asm](wasm.asm)   | Generates a complete **WASM** module from scratch.                |

Due to the high effort required to create and maintain these examples, We welcome contributions from the community. If
you have an idea for a new example or improvements to existing ones, please submit a pull request or open an issue.
